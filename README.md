# metablazt

Magic the Gathering Card Database and Deck builder.

## Project Structure

- `/web` – Next.js App Router frontend (TypeScript, Tailwind) configured for static export.
- `/docs` – architecture references (`schema-and-events.md`, `design-tokens.md`).

## Getting Started

```bash
cd web
npm install
npm run dev
```

## Static Export Preview

```bash
cd web
npm run build
# Static output available in web/out for GitHub Pages deployment
```

Refer to `ROADMAP.md` for current Phase 0/1 milestones.
